Справочник «Пользователи»

Справочник «Пользователи» предназначен для хранения списка пользователей. В основном это пользователи, работающие с конфигурацией (пользователи ИБ).

Идентификация пользователя ИБ с пользователем справочника выполняется по соответствию имени пользователя ИБ имени пользователя справочника.

Информация о всех действиях пользователя при работе с информационной базой отражаются в журнале регистрации. Журнал регистрации вызывается из пункта меню «Сервис» – «Журнал регистрации». Подробное описание порядка работы с журналом регистрации приведено в книге «Руководство пользователя».

Интерфейс

Пользователям можно назначить Профиль, указать Физическое лицо, которое соответствует данному пользователю.

Для редактирования настроек пользователя необходимо нажать кнопку "Настройки пользователя".

Для изменения групп, в которые входит пользователь необходимо нажать кнопку "Группы пользователя".

Для редактирования параметров пользователя ИБ необходимо нажать на кнопку "Пользователь ИБ".

На закладке "Контактная информация" вводится информация о контактной информации пользователя - адрес пользователя, телефон, адрес электронной почты и т.д. Эта информация используется при работе пользователя с клиентами и при работе со встроенной электронной почтой.

Создание новых пользователей

Только Администратор пользователей может создавать, изменять и удалять пользователей.

Создание пользователей ИБ

Создание пользователей ИБ можно выполнять в режиме конфигуратора или в режиме предприятия.

Создание пользователя ИБ в режиме предприятия выполняется в форме "Пользователь информационной базы", которая открывается из формы справочника с помощью команды "Пользователь ИБ" подменю "Администрирование".

Полномочия пользователей

Полномочия пользователей определяются его ролями и дополнительными правами.

Полномочия можно назначать при помощи профилей.

Права доступа на уровне записей определяются группами пользователей, в которые входит пользователей.
